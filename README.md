## DHT11(온도 습도센서)

- DHT11는 회로가 복잡하지 않으며, 아두이노와 연결하기도 쉬워 아두이노의 기초를 배우거나 프로젝트를 진행 때 자주 사용되는 온도, 습도 센서입니다.
- 센서 자체의 측정 범위는 매우 높지는 아니지만 일반적인 상황 (집안의 온도를 측정하는 등)에서 사용하기는 적합한 제품입니다.
- 측정 습도 : 20~95%
- 측정 온도 : 0 ~ 50℃
